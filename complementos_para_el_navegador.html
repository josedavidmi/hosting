Complementos para el navegador

Para los navegadores más usados

Extensiones para Chrome

  • GPT for Slides. Esta extensión para Chrome, crea una presentación de Google a partir de un tema, un texto, un vídeo de YouTube o una URL. En su versión gratuita permite hacer 3 presentaciones mensuales.
  • GPT for GMAIL. Escribe correos electrónicos para Gmail a partir de un asunto o resumen.
  • DocGPT - AI Writer for Docs. Escribe texto en Google Docs a partir de un título o texto.
  • Speak Subtitles for YouTube. Lee los vídeos en el idioma que se le indique.

Extensiones para Firefox

  • Youtube Transcript AI Summary. Hace resúmenes de los vídeos de YouTube mientras se están viendo.
  • ChatGPT for Google. Muestra las respuestas de ChatGPT durante las búsquedas.
